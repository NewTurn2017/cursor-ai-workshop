# 실습 1: 나를 소개하는 홈페이지

> **소요 시간**: 35분
> **기술**: HTML/CSS/JS, 반응형
> **배포**: Vercel

## Step 1: 기획

Agent에 아래 프롬프트를 입력하세요:

```text
나를 소개하는 개인 홈페이지를 만들어줘.
한 페이지짜리 반응형 웹사이트로, 깔끔하고 현대적인 디자인으로 부탁해.
```

AI가 계획을 보여주면 → 검토 → 수정 요청 또는 승인 → 구현 시작

## Step 2: 수정

원하는 대로 다듬으세요. 여러 개를 조합해도 됩니다.

```text
헤더 배경색을 네이비(#1a1a2e)로 바꿔줘
```
```text
프로필 사진 영역을 상단에 원형으로 추가해줘
```
```text
스크롤 시 부드럽게 이동하는 네비게이션을 추가해줘
```
```text
다크모드/라이트모드 토글 버튼을 넣어줘
```
```text
푸터에 GitHub, LinkedIn 아이콘 링크를 추가해줘
```
```text
전체적으로 더 모던하게 리디자인해줘
```

### Browser Tool 활용

Cursor 내장 브라우저로 시각적 수정도 가능합니다:
- **컴포넌트 클릭 → 프롬프트로 수정**: 브라우저에서 요소를 직접 클릭 후 지시
- **CSS 실시간 미리보기 → Apply**: 프롬프트 없이 CSS를 시각적으로 수정

## Step 3: 배포

```text
이 프로젝트를 Vercel로 배포해줘.
vercel CLI로 배포하고, 배포된 URL을 알려줘.
```

## 체크리스트

- [ ] 홈페이지가 로컬에서 잘 보인다
- [ ] 수정 요청을 2번 이상 해봤다
- [ ] Vercel 배포 URL이 나왔다
- [ ] 모바일에서도 확인했다
