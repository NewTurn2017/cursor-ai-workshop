# Mock Data — 실습용 샘플 데이터

이 폴더의 데이터는 워크숍 실습에서 사용하는 **가상 데이터**입니다.
실제 고객 정보나 민감한 데이터가 아닙니다.

## 파일 목록

| 파일 | 용도 | 실습 |
|------|------|------|
| `incident-briefing.json` | 업무 브리핑 허브 샘플 | 실습 1 |
| `security-events.json` | 요청 정리 보드 샘플 | 실습 1 |
| `playbooks.json` | 단계별 조치 보드 샘플 | 실습 2 |
| `incident-timeline.json` | 업데이트 타임라인 샘플 | 실습 2 |
| `outage-notice-input.json` | 공지/FAQ 생성기 입력 샘플 | 실습 2 |
| `nginx-sample.conf` | 설정 설명기 샘플 | 실습 3 |

## 사용 방법

Cursor에서 이 폴더를 열고, 실습 가이드의 프롬프트를 따라가면 됩니다.

예:
```text
이 JSON 데이터의 구조를 정리하고, 핵심 항목을 요약해줘.
```
